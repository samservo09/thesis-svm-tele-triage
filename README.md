# thesis-svm-tele-triage
This repository contains all the code files required to run simulated existing and proposed algorithm of Support Vector Machine for the research, "Modified Support Vector Machine for Text Classification applied for Tele-triage".

The implementation of SVM on text-classification is currently based on [this](https://medium.com/@bedigunjit/simple-guide-to-text-classification-nlp-using-svm-and-naive-bayes-with-python-421db3a72d34) site.

This is subject to change as it should be from a research paper.
